# Server Side Rendered App with Next.js
A Next.js App to create server side rendered app with mock data of space X mission details. 

## Getting Started
### Dependencies
* Next.js
* npm, node.js
### Installing

 ```
npm run install
```

### Executing program
* Run the command
```
next dev
```
* Or alternatively, run
```
npm run start
```
